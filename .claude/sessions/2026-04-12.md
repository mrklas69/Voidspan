### Herní koncept — rozhodnuté základy

| Téma | Rozhodnutí |
|---|---|
| Médium | Digitální hra, webový prohlížeč, 8-bit retro grafika |
| Interakční model | PvPvE sandbox (kompetitivní + kooperativní + solitaire vrstvy) |
| Perzistence | Perzistentní svět, běží věčně |
| Čas | Real-time (wall clock) |
| Geometrie | Binární strom zakořeněný v přístavu (~10 políček), expanze dvěma směry, další větvení rekurzivně |
| Entropie | Neopravovaná políčka chátrají, pás se může trhat → izolované ostrovy claimů |
| Větvení | **Serverová událost**, ne automatika. Drahé, kolektivní úsilí. Vzácné a pamatované. |
| NPC-správce | Offline proxy, jen údržba (ne ofenziva). Náklady = CPU/palivo. Úrovně programování: presety → bloky → skript (k rozhodnutí) |
| Ekonomika | Tři zdroje: **Echo** (život/palivo), **Kredo** (stavba), **Rift** (rizikové akce, jen z forků). Monopoly + Ponzi dynamika na vlastnictví políček. |
| Instituce | Katastrální úřad, soud, banka, šerif — kolektivní nedestruktibilní políčka v hubu. Řeší pozůstalosti, spory, zákon. |
| Questy | Hub nabízí práci (pomocník šerifa = mzda za vymáhání práva). Greenhorn onboarding + alternativní příjem proti riziku claimu. |
| Vítězství / prohra | K rozhodnutí (perma vs. reinkarnace, legacy body, Jádro…) |
