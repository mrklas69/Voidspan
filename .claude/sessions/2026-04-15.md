## UI Layer Stack axiom (S19, hlavní téma)

Vrstevnatý z-order od dekorace k nejmodálnějšímu prvku. Izomorfní: výš = interaktivnější + naléhavější.

- **0 Hvězdné pozadí** (drift, layer 0)
- **1 Kosmické sprity** (asteroidy, kapsle, dopravní)
- **2 BELT** (segment, epicentrum — vertikální growth nahoru/dolů)
- **3 Ukotvené panely Top/Main/Bottom** (texty bez pozadí, zvažuje se průhledný panel)
- **3.5 Floating workspace** (K/U/Z/E/P — parkoviště, layer 4 odložen)
- **4 Link-triggered overlays** — kontextové, ne striktně hierarchické:
  - 4.1 Infotip (plaintext, autoclose)
  - 4.2 Karta (ASCII+barvy, autoclose)
  - 4.3 Popover (8bit wiki, autoclose)
- **5 Modální okna** (Close + ESC, **nezastavují čas**):
  - 5.1 info + Close (Help, Win, Loss, Mapa)
  - 5.2 velké přehledy (Výzkum, Obchod, Politika)
  - 5.3 Nastavení
  - 5.4 Chat s hráči, hlasování

**Axiomy:**
- ESC = globální bezpečný odchod (všechny vrstvy).
- `#hull-dark` jediné povolené pozadí overlay textů, mid/light varianty pro gradient 4.1→4.3.
- Texty ukotvených panelů bez pozadí.
- Modální nezastavují čas.

## Průhlednost boxů — Censure S19 + záhada

**Censure!** Pozadí dialogů bylo solid (nebo stejná barva jako canvas). User vyhlásil axiom: **všechny UI boxy mají transparentní pozadí, ladíme alpha per-box typ**. Memory zápis: `feedback_dialog_bg_transparent.md`.

### Bug: alpha vypadala binárně (0 nebo 100 %)
**Root cause:** `UI_PANEL_BG` = `COL_VOID_BLACK` = `#0a0a10` = stejná barva jako canvas backgroundColor. Alpha tedy neměla kontrast proti čemu projít — jediný viditelný rozdíl byl nad hvězdami. Fix: panel bg → `COL_HULL_DARK` (`#1a1e28`, slot 02).

### Border rect → stroke
Samostatný solid `border` rect pod bg zakrýval hvězdy. Fix: `setStrokeStyle(1, UI_BORDER_DIM)` přímo na bg rect.

### Ladění alpha
Několik iterací (0.88 → 1.0 → 0.4 → 0.2 → 0.5 → 0.7 → 0.8 → **0.9**). Finální hodnota: **panel alpha 0.9 (10% průhlednost), overlay 0.25**.
