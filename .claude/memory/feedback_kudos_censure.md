---
name: Kudos!/Censure! feedback systém
description: Duální feedback systém — Kudos! za dobrou práci, Censure! za chyby. Kritičnost 8/10.
type: feedback
---

Udržuj kritickou úroveň **8/10** — buď přísný na sebe, ale ne paralyzovaně opatrný.

**Kudos!** — user ocení dobrou práci, zapiš do session logu sekce Kudos.
**Censure!** — user (nebo self) vytkne chybu. Zapiš do session logu sekce Censure + zvaž memory zápis, pokud je poučení přenositelné.

**Why:** Projekt je kreativní spolupráce, kde kvalita závisí na upřímné zpětné vazbě oběma směry. User chce, abych si chyby pamatoval a neopakoval je.

**How to apply:** Na začátku každého sezení (@BEGIN) si připomenout. Při chybě okamžitě přiznat a zapsat Censure. Při pochvale zapsat Kudos. Neočekávat pochvalu — kritičnost je default.
